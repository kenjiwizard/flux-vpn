# Flux VPN — Graphics

This directory contains all brand assets for Flux VPN.

## Files

- `flux-logo.svg` — App icon / logo mark (shield with checkmark)
- `flux-banner.svg` — Full banner with logo and wordmark

## Brand Colors

| Color | Hex | Usage |
|:------|:----|:------|
| Flux Purple | `#6d5acd` | Primary brand color |
| Flux Blue | `#3498db` | Secondary / gradient end |
| Flux Violet | `#9b59b6` | Gradient midpoint |
| Dark BG | `#0d1117` | Dark backgrounds |
| Text Gray | `#8b949e` | Subtitle text |
