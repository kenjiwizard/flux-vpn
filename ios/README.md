# Flux VPN — iOS

The iOS client for Flux VPN, built with Swift and SwiftUI.

## Prerequisites

- Xcode 15+
- iOS 17.0+ deployment target
- macOS Sonoma or newer

## Building

1. Open `ios/FluxVPN.xcodeproj` in Xcode
2. Select your target device or simulator
3. Build & run (⌘R)

## Architecture

The iOS app is a standalone SwiftUI implementation with a Network Extension for tunnel management.
